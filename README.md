# Lumière 🎬

**«Що подивитись сьогодні»** — iOS-додаток, що вбиває параліч вибору фільму.
Задаєш настрій, жанр і скільки маєш часу — Lumière пропонує один влучний фільм.

> Showcase-проєкт (iOS / SwiftUI).

## Можливості (в розробці)
- 🎯 Підбір фільму за настроєм / жанром / тривалістю
- 🃏 Картка-пропозиція: «Беру / Інший / У список»
- 📥 Watchlist офлайн
- 📊 Історія рішень + статистика смаку
- 🔍 Пошук фільмів

## Стек
- **Swift, SwiftUI**
- **SwiftData** — офлайн-зберігання
- **async/await + URLSession** — TMDB API
- **MVVM + @Observable**
- **Swift Charts**
- **Unit-тести**

## Статус
🚧 В активній розробці.
